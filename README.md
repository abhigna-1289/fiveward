# fiveward
## Free. Focused. Fiveward.

fiveward contains various study materials for AP classes and exams, helping students spend their time studying instead of searching for the right resources.

## Live Site
[fiveward.vercel.app](https://fiveward.vercel.app)

## What is fiveward?
fiveward is a free study tool for AP classes and exams. It provides efficient, research-backed study methods used by high scoring students, all in one place, with no paywalls and no missing content.

## Why I Built It
When I was studying for my AP classes and exams, I tried numerous resources and methods to find what worked best. But I kept spending more time finding the right way to study than actually studying. So I built fiveward to fix the issues I kept running into, hidden paywalls, missing content, and resources that weren't aligned to the College Board curriculum. I believe every student should have access to the right resources without wasting time searching for them.

## Features
- **Flashcards** - mark cards as "got it" or "still learning", shuffle deck, restart anytime, and switch term and definition
- **Study Guides** - covers all content aligned with the College Board CED
- **Practice Questions** - MCQ, FRQ, and mixed mode with explanations after each answer; FRQs include a rubric for self-grading
- **Progress Tracker** - tracks completion, study time, daily streaks, average scores, points, and leaderboard rankings
- **Study Mode Picker** - choose to study by unit or individual topic, for flashcards, practice questions, or study guides
- **Study Timer** - set session timers to stay on track

## Tech Stack
- **HTML, CSS, JavaScript** - frontend
- **Python Flask** - backend server
- **Supabase** - database and authentication
- **Vercel** - frontend hosting
- **Render** - backend hosting
